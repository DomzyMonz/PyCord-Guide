Non Official tutorial for PyCord

PyCord is a modern, easy to use API wrapper for Discord and a good alternative for Discord.py.

Steps:

First You need to uninstall discord.py

1. pip uninstall discord

2. pip uninstall discord.py

3. pip install py-cord

Now you have sucessfully installed py-cord
